# MyEshop 

یک پروژه‌ی فروشگاه اینترنتی ساده برای تمرین و یادگیری مفاهیم برنامه‌نویسی و توسعه‌ی وب.

##  ویژگی‌ها
- مدیریت محصولات (افزودن، ویرایش، حذف)
- نمایش لیست محصولات برای کاربران
- سبد خرید ساده
- سیستم ثبت‌نام و ورود کاربران
- طراحی ماژولار و قابل توسعه

##  تکنولوژی‌های استفاده‌شده
- **C# / ASP.NET Core** برای بک‌اند
- **Entity Framework Core** برای مدیریت دیتابیس
- **SQL Server** به عنوان پایگاه داده
- **HTML / CSS / Bootstrap** برای رابط کاربری
